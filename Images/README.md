# Images

Project photos, renders, and progress pictures.

## What to add here

- Photos of the physical build at each stage
- Renders exported from `CAD/Images/`
- Screenshots of the Automation Studio HMI
- Any promotional or presentation images

## Naming Convention

```
<YYYY-MM-DD>_<Description>.<ext>
```

Examples:
- `2024-03-15_AssemblyFront.jpg`
- `2024-04-02_WiringComplete.jpg`
- `2024-05-10_FinalRender.png`
